４。けいざい（Economy）
  - おもに　２つ　あります。
    - のうぎょう（Agriculture）
      - 日本では　せいだい（largest）の　リンゴを　そだてて　いる　ちほう。
      - そして、　赤スグリ　や　ガーリック　や　他のくだもの　なども　そだてて　います。
    - かんこうぎょう（Tourism）
      - 青森には　きれいな　けしきも、ほうふな（abundant）文化も、ながい　れきしも　あるし、それで　
        かんこうぎょうに　人気のところ　です。
